Data Storage

Short Summary on the topics of Data Storage and Data Manipulation

Data Storage is about various digital information storing methods. Data can be stored on multiple media, including hard drives, flash drives, and optical discs (Luo, 2021). The type of media used will typically depend on stored data and the desired access time and durability. Today's enterprises and customers need data storage to satisfy their high-level computing needs, such as artificial intelligence (AI), big data projects, IoT, and machine learning. On the other hand, vast data storage volumes are essential for preventing data loss due to accidents, errors, or fraud. Organisations may also use data storage as a backup option to avoid data loss.

References:
Luo, C. (2021) Computer Data Storage and Management Platform Based on Big Data. In Journal of Physics: Conference Series (Vol. 2066, No. 1, p. 012022). IOP Publishing.

Data Manipulation

Data Manipulation entails the process of structuring data to make it simpler to interpret, more designed, or more organised. Data manipulation is a critical element of organisational processes and optimisation. You need to know how to deal with data and turn it into useful information by analysing trends, consumer behaviour, and financial data. The language used in data manipulation is DML (Data manipulation language) (Abu Ghali and Abu-Naser, 2019). It is a programming language that modifies data in a database by inserting, updating, and deleting data to filter or map it.

References:
Abu Ghali, M.J. and Abu-Naser, S.S. (2019). ITS for Data Manipulation Language (DML) Commands Using SQLite. dstore.alazhar.edu.ps. Available from: http://dstore.alazhar.edu.ps/xmlui/handle/123456789/101 [Accessed 31 May 2022].

Operating Systems

OS is software that serves as a translation or intermediary system between application programs and hardware installed on the computer (Szopa, 2021, 340). Put another way; you cannot directly utilise software applications with computer hardware if you do not have a medium that links between them. It controls the computer's memory and operations and its hardware and software components. Additionally, it allows you to interact with the computer. The most common OS are MacOS, Linus, and Microsoft Windows.

Networking and the Internet

Networking is the study of how computers can be connected to facilitate data exchange. On the other hand, the Internet is a vast network that connects different computers worldwide (Comer, 2018). This topic provides an overview of networking protocols and the Internet by explaining various vital concepts and terms. Networking technology has transformed the universe and established a new arena for each nation's overall development. The Internet system provides multiple information services and resources, such as the World Wide Web (WWW), telecommunications, electronic mail, etc. It employs typical internet protocols like HTTP, TCP/IP, etc.

References:
Comer, D.E. (2018) The Internet book: everything you need to know about computer networking and how the Internet works. Chapman and Hall/CRC.

Algorithms

An algorithm is âa process or set of rules to be followed in calculations or other problem-solving operationsâ (Royston, 2018, 47). In other words, it is a set of instructions that outline a step-by-step procedure of executing a problem to get the desired results. The Algorithms are language-independent, which means that they may be implemented in any language and still provide the same results as anticipated. There are many different types of algorithms, and they can be classified in several ways. Some typical classification schemes include the kind of problem they solve (e.g., searching, sorting, or graph algorithms), the type of data they operate on (e.g., strings, numbers, or arrays), or the type of operation they perform (e.g., addition, multiplication, or comparison). Algorithms can be implemented in various ways, including a set of instructions in a programming language, a logic circuit, or a physical device.

Programming Languages

Programming languages are about the definition of computer programs and different types of languages. A computer program is a set of instructions that computers use to accomplish a particular task. It must adhere to a specific programming language. Therefore, the programming language is a combination of grammatical rules (syntax) and vocabularies that instruct a computer to accomplish specific tasks (Sobral, 2020); for a computer to understand a programming language, it needs to be compiled and then interpreted. Compile converts a high-level programming language program to object code from source code. A compiler does this. The interpreter executes high-level language instruction by instruction. There are several programming languages used to code various computer programs. Languages such as PHP, Javascript, Python, Java, etc., can be used to develop websites. Where else C++ is used to create games.

References:
Sobral, SR (2020). 30 Years of CS1: Programming languages evolution. Uportu.pt. DOI:http://hdl.handle.net/11328/2925.

Software Engineering

Software engineering is the field that studies how theory, practice, and knowledge can be used to build secure software systems that meet the computing demand of consumers and users (Heeager and Nielsen, 2018, 28). It may be implemented in computer systems and organisations of any size, from the most basic to the most complex. Software engineering uses engineering methodologies, procedures, techniques, and evaluations. Whether done by a person or a team, software development necessitates selecting the best tools, methods, and approaches for the situation.

Abstraction hides the details of a complex system and exposes only the essential features (Zhang et al., 2019, 188). Database Systems

A database system is a collection of software used to store, manage, and retrieve data from a database (Kraska et al., 2021). It typically consists of a database management system (DBMS), a database server, and a database client. In this topic database management system is explained in detail. DBMS is a tool that collects electronic and digital data to retrieve and store important information. A database server is a computer that stores the data in a database. A database client is an application for accessing the database data. Artificial Intelligence

AI is a broad topic that focuses on developing intelligent computers able to do activities that typically require human intelligence (Jordan, 2019). 2. Business-Business has deployed AI methodologies and tools to solve problems.
3. Programming- AI comprises symbolic programming, search, and problem-solving knowledge. AI programming languages are LISP, PROLOG, and object-oriented languages such as Python, C++, etc.
Applications of AI include speech recognition, game playing, computer vision, natural language, heuristic classification, expert systems, consumer marketing, intrusion detection, machine translation, identification technologies, etc.
