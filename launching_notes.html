Data Storage and Data Manipulation

Short Summary on the topics of Data Storage and Data Manipulation

Data Storage

• Boolean operations combine true/false values and are a combination of AND/OR/XOR operations, where there is an input, an output and an operation that occurs.
• XOR produces an output of True when both inputs are different.
• NOT operations produce an opposite result as output.
• Physical hardware is what processes the binary data that is used.
• Hexadecimal Notation is used to represent bits in multiples of four.
• Strings of eight bits is called a byte and represents a cell.
• Cells are commonly envisioned as rows, with the left part being the most significant bit and the rightmost cell being referred to as the least significant bit.
• Computers’ main memory stores different cells that can be accessed when required; each cell is accessible by itself, thus referring to a RAM (Random Access Memory).
• Storage systems such as USB drives are generally slower because it is an external memory, unlike a computer’s main memory that stores and retrieves data faster.
• Unicode holds a unique pattern of up to 21 bits when combined with UTF-8 (Unicode Transformation Format 8-bit representing a symbol.
• Numeric Information is commonly stored as binary notation.
• Musical Instrument Digital Interface (MIDI) is an encoding system representing sound.
• Binary Addition is similar to regular addition, where the one is carried over in place of a value.
• Overflow or errors can occur in some programming languages due to bits that represent characters.
• The LZW technique is used for encoding by using basic building blocks, adding and making up a dictionary.
• GIF is a method used to compress an image using the LZW technique; it’s a method where the colours are stored up to 256 pixels, afterwards stored in a dictionary, forming a byte for each pixel.
• Parity Bits is a method used to detect errors that are used to detect misinformation or loss of data when transferred and processed by finding the odd patterns.

Data Manipulation

• A CPU consist of an Arithmetic/Logic Unit, Control Unit and Memory with the Control Unit as the ‘Brains’ of the CPU that sends signals between other units by taking input and then producing outputs.
• I/O Instructions are generally considered part of the data transfer group, where it transfers commands to communicate with devices outside a CPU.
• Control is responsible for the execution of instructions.
• Instruction Registers and Programs Counters are two special-purpose registers within a CPU; responsible for executing a set of instructions in the correct order.
• A Program Counter is used to keep track and execute a set of instructions given, performing the tasks required in order.
• Circular shifts occur when the bit at the right end falls off, it gets replaced on the left side. Logical shift is when the bit is discarded and replaced with a zero. The arithmetic shift is when the bit is not changed.
• Controllers are used to communicate between a CPU and other devices; by translating signals/messages between the CPU and the external device. Typically, controllers would be attached to the bus connected to the CPU.
• Direct memory access (DMA) is when an external device is connected to the Bus and directly to the memory, thus accessing the main memory.
• Pipelining is a technique used to allow a faster computer throughput; by allowing steps within a machine cycle to overlap.

Network, Data Mining, Animation and Artificial Intelligence

Short Summary on the topics of Network, Data Mining, Animation and Artificial Intelligence

Networking

• PAN (Personal Area Network) is commonly used for short-range connections.
• LAN (Local Area Network) is commonly used to connect computers found within the same building.
• MAN (Metropolitan Area Network) Cover a more comprehensive connection, such as within a Town.
• WAN (Wide Area Network) covers more extensive connections, such as worldwide.
• A Bus is a common topology that connects multiple machines into one line.
• A hub is a device that replicates the behaviour of a Bus connecting machines together.
• Carrier Sense, Multiple Access with Collision Detection (CSMA/CD) is a protocol that ensures that each message will be transmitted to all machines within the network. Sometimes, messages can collide; thus, the message has to be retransmitted.
• A bridge connects two buses by sending messages across a bus if the machine is registered to receive that message.
• A Switch acts similarly to a bridge but connects more than two buses.
• A router connects incompatible networks such as an Ethernet network and a Wifi network.
• When many computers work together mimicking the power of a supercomputer, it is referred to as cluster computing.
• Internet service providers are responsible for providing connected networks forming the Internet.
• Assigning machines to IP Addresses enables them to connect to the Internet, which is made of bit patterns. While another form is a domain that can be used easily by humans.
• Domain Name Systems (DNS) are directories that translate domain names to IP addresses.
• Post Office Protocol version 3 (POP3) and Internet Mail Access Protocol (IMAP) are two standard protocols used to access mails.
• Brower’s and Webservers are two methods used to allow users to access hypertext or Hypertext Markup Language (HTML) on the Internet by using URLs.
• XML encodes either images or other data types in another format that can be unified, standardising data in another form.
• The internet software layers consist of four layers: Application Layer, Transport Layer, Network Layer, and Link Layer.
• Transmission Control Protocol (TCP) is a method that has been used to implement the Internet software layers and is generally considered to be a less efficient method than User Datagram Protocol (UDP).
• IPv4 was commonly used to implement the four layers, but due to consistent growth and demand, IPv6 was then implemented to handle more devices routing.

Data Mining

• Class description is a data mining method that provides a way to generalise data by characteristics. Class discrimination is another form of data mining that divides two groups to provide the required data. Cluster Analysis discovers properties that would give new groups. Association Analysis is a method where points between data would be identified. Outlier Analysis finds data that wouldn’t be categorised in a regular pattern. Sequential Pattern Analysis is a method to find ways to correlate data over a set period.
• Security and Privacy Concern is considered a high impact of database technologies.
• When privacy concerns aren’t relevant, data can be exploited between databases.

Animation

• Frames is the primary form of presenting animations by displaying multiple frames that would provide the animation itself.
• Dynamics and Kinematics are methods of describing motion that would be present in 3D animation. Dynamics deals with the physics of how an object should move, while Kinematics deals with the parts of the object on how they corollate and move appropriately.

Artificial Intelligence

• Artificial Intelligence can be dated back to 1950 when Alan Turing presented an article “Computing Machinery and Intelligence” that would demonstrate how machines can be programmed to mimic humans and present intelligent solutions.
• Artificial Intelligence deals with Image processing and Language processing to handle data as an input.
• Neural Networks is basics of how Artificial Intelligence operates, by following a similar pattern to an actual brain; working by receiving input and then processing that input by adding weights to the input to compare it with a certain threshold and then there would be output.
• Hidden layers are present within a neural network. They offer the purpose of presenting weights and processing the data; depending on the number of hidden layers, data can be processed more by increasing the number of hidden layers.
• Training Neural Networks include processing the information within the hidden layer and adjusting the weights accordingly to process the data many times over.
• Artificial Intelligence can ultimately support humans in many aspects of repetitive tasks and working non-stop. However, there is always the concern of program execution where providing the Machine with a way to execute commands could be devastating since errors can occur with Artificial Intelligence. Thus, ethical and legal concerns emerge.
